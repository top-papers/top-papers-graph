# 12-недельный план (case study IED: fast-charge Li-ion)

Этот файл — “операционный план” для squad:  
1 ML-инженер + 3–6 экспертов (электрохимия/мат.моделирование/силовая электроника).

## Недели 1–2: Baseline + reasoning trajectories
- Dev:
  - поднять инфраструктуру (docker compose),
  - настроить LLM,
  - сделать baseline симуляцию PyBaMM (CC/CCCV-приближение), сохранить результаты.
- Experts:
  - выбрать 2–3 под‑темы (plating, SEI, temp effects, protocol design),
  - заполнить 2–3 reasoning trajectories (E1).

## Недели 3–5: GraphRAG + граф‑ревью
- Dev:
  - загрузка/парсинг статей, извлечение триплетов, KG,
  - GraphRAG запросы под research question.
- Experts:
  - E2: проверка графов, уточнение условий применимости.

## Недели 6–8: Дебаты агентов + чек‑лист Скептика
- Dev:
  - внедрить экспертные критерии в промпт Скептика,
  - сохранять логи дебатов.
- Experts:
  - собрать чек‑лист “Reviewer #2 для fast‑charge”,
  - начать E3 (рецензии гипотез).

## Недели 9–10: Проверка гипотезы в PyBaMM + абляции
- Dev:
  - реализовать 2–3 профиля зарядки,
  - абляции: без скептика / без графа / без экспертных сигналов.
- Experts:
  - рецензии гипотез + комментарии к интерпретации.

## Недели 11–12: Статья + pre-submit audit
- Dev:
  - фигуры/таблицы, финальные результаты,
  - прогнать `scripts/paper/pre_submit_audit.py`.
- Experts:
  - финальная вычитка, заполнение disclosure/autonomy log.
