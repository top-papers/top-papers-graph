# Учебные материалы курса (12 недель)

В папке `course/weeks/` лежат 12 Markdown-файлов — по одному на каждую неделю.

Рекомендованный порядок:
- Week 01 → Week 12 (каждая неделя добавляет один слой в общий end-to-end пайплайн)

Быстрый старт:
```bash
./scripts/bootstrap.sh
top-papers-graph demo-run
```

Дальше на каждой неделе вы делаете маленькое улучшение проекта (код/конфиг/артефакты разметки),
которое повышает качество темпорального графа знаний и гипотез.
