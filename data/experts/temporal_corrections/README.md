# Temporal corrections

Цель: исправлять временные привязки утверждений (Temporal KG) и улучшать извлечение времени.

Используется, когда:
- модель поставила неверный год/период
- утверждение относится к диапазону (например, 2016–2019)
- время присутствует только в подписи/таблице (multimodal)

См. `_template.json`.
